# Projeto CadVeículos - Atitus Faculdade

Sistema desenvolvido para a disciplina de Algoritmos e Programação. O objetivo do projeto é gerenciar o cadastro de veículos utilizando Java e conceitos de Orientação a Objetos.

## Funcionalidades Implementadas
1. Cadastro de veículos com validação de duplicatas (Case Insensitive).
2. Busca por nome com exibição do total de registros.
3. Listagem completa exibindo o número total de veículos.
4. Mensagens de alerta para listas vazias.
5. Edição de nomes de veículos existentes por índice.
6. Ordenação alfabética manual (Bubble Sort) sem uso de bibliotecas externas.
7. Remoção de veículos por índice e por nome.

## Como Executar
O projeto é composto por três arquivos principais:
* `Veiculo.java`: Classe que define o objeto veículo.
* `Input.java`: Classe utilitária para leitura de dados via teclado.
* `CadVeiculos.java`: Arquivo principal que contém o menu e a lógica do sistema.
